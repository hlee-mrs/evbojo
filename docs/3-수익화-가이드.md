> 핵심 결론: **월 100만원은 광고 단가가 아니라 트래픽 싸움**입니다. 자동차·금융 인접 니치의 애드센스 RPM(1,000뷰당 수익)은 대략 3,000~5,000원 → **월 20~35만 페이지뷰**가 필요해요. 그래서 이 사이트는 SEO 구조(지역·차종별 URL, FAQ 스키마, 기준일 스탬프)를 처음부터 깔아뒀습니다.

## 0. 지금 상태
- 광고 슬롯 11개가 이미 배치되어 있음 (홈2, 지역2, 차종2, 계산기1, 진단1, 환수1, 가이드/법령/FAQ 각각)
- 모두 **고정 높이(CLS<0.1) + "AD" 라벨 + 입력폼과 24px 이상 간격** — 애드센스 정책(오클릭 유도 금지) 준수 설계
- 현재는 `enabled:false` — 자리만 잡혀 있고 아무것도 로드 안 함

## 1. 시작 순서 (0~2개월)
1. **카카오 애드핏 먼저** (심사 빠름, 국내 모바일 채움률 좋음)
   - adfit.kakao.com 가입 → 매체 등록 → 심사(수일)
   - 승인 후: `site/assets/js/app.js`의 광고 로더를 애드핏 스크립트로 교체하거나 슬롯별 삽입
2. **애드센스 신청 조건 갖추기**
   - 콘텐츠: guide/law/faq 등 텍스트 페이지가 이미 있음 → 여기에 **가이드 글 15~20편**을 추가하면 승인 확률이 크게 올라요 (계산기·표만 있는 사이트는 "가치 낮은 콘텐츠"로 반려되는 단골 유형)
   - privacy.html(제3자 광고 쿠키 고지 포함) ✅ 이미 있음 / about.html ✅ / 문의 이메일 ✅
   - 커스텀 도메인 + HTTPS ✅ (도메인 가이드)
3. **애드센스 승인 후 스위치 켜기**
   - `site/assets/js/app.js` → `SITE.ads = { enabled:true, client:'ca-pub-본인ID' }`
   - 각 페이지 슬롯의 `data-ad-slot` 번호 입력 (애드센스에서 광고 단위 생성)
   - `site/ads.txt`의 주석 해제 + 본인 pub ID 입력
   - 자동광고는 **모바일 앵커만 ON**, 전면(비네트)·사이드레일 OFF 권장 (이탈 방지)

## 3. 광고 외 수익원 (갭 필러)
- **쿠팡 파트너스**: 가이드 글에 "집밥 필수템(완속충전기·케이블·홀더)" 자연 삽입. 링크에 `rel="sponsored nofollow"` + "파트너스 활동으로 수수료를 받을 수 있음" 고지 필수
- **보험·장기렌트 제휴(CPA)**: 실구매가 결과 아래 배너 1개 — 리드당 수천~수만원(광고 클릭 수십 건 가치). 단 "추천/상담" 화법은 금소법 리스크 → 공식 배너 링크아웃만
- 절대 금지: 광고를 콘텐츠로 위장, "클릭해주세요" 유도, 계산 버튼 옆 광고
